[HYBRID DATA CENTER (AccountingRepository.java)]
- Online: Fetches fresh data via `POST /sync` from FastAPI Server, performs write-through local cache update in Room DB, serves updated data.
- Offline: Swaps seamlessly to local Room SQLite cache without lag, records actions to `sync_queue`, schedules WorkManager.

[ROBUST SYNC HANDSHAKE (LWW Conflict Resolution)]
- Client pushes `client_logs` payload from `sync_queue` to server `/sync`.
- Server uses Last-Write-Wins (LWW) timestamp comparisons on SQLAlchemy entities, applies changes, and extracts server updates since client's last sync time.
- Client processes server updates, bulk-merges them, and clears synced items from `sync_queue`.

[SELF-HEALING & AUTOMATED DIAGNOSTICS (AppHealthMonitor.java)]
- Asynchronous DB check: runs integrity queries, detects and purges orphaned records (e.g. `invoice_items` with missing parent document ID), checks sync queue length.
- Latency check: evaluates network ping to cloud `/health/check`.
- Pushes dynamic client telemetry logs to server via `/telemetry/report` in case of anomalies.
